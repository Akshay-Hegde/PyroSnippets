h1. PyroChunks

h2. Description

Now updated for PyroCMS 1.x.

Module for "PyroCMS":http://pyrocms.com by "Phil Sturgeon":http://philsturgeon.co.uk/. Allows users to create and manage small chunks of reusable HTML or text content for their site. Useful for giving clients access to small chunks of content that are not page content.

h2. Author

* "Adam Fairholm":http://twitter.com/adamfairholm
* "Stephen Cozart":http://twitter.com/stephencozart - Updated to PyroCMS 1.x

h2. Installation

To install, just upload the zip file using PyroCMS's built in module upload tool. The "chunks" MySQL table will be added.

h2. Usage

After installing if you are using a PyroCMS version less than 1.1, autoload the Chunks library (chunks/Chunks). This will allow access to chunks in your page layouts.
If you are using PyroCMS 1.1.x you will NOT need to add the chunks library to your config/autoload array.
